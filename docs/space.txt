空間判定処理

空間 A[As, Ae]  B[Bs, Be] (As < Ae, Bs < Be)
(Ae, Beは, roundup(AsまたはBs+length, page-size) -1)
空間Xがアドレスnを含むとは
  ( ( Xs <= n ) && ( n < (Xe + 1 ) ) )が真であること

(1) AよりB全体が前にある場合
    As > Be,                  かつ
    Aの先頭をBが含んでおらず, かつ, 
    Aの末尾をBが含んでおらず, かつ, 
    Bの先頭をAが含んでおらず, かつ
    Bの末尾をAが含んでいる
    

(2) Aの先頭にBの末尾が接している場合
    As > Bs,                かつ,
    Aの先頭とB末尾+1が等しく, かつ, 
    Aの末尾をBが含んでおらず, かつ, 
    Bの先頭をAが含んでおらず, かつ
    Bの末尾+1とAの先頭が等しい(重複条件)

(3) Bの末尾がAに含まれる場合
    Aの先頭をBが含んでいて, かつ, 
    Aの末尾をBが含んでおらず, かつ, 
    Bの先頭をAが含んでおらず, かつ
    Bの末尾をAが含んでいる

(4) AとBの先頭アドレスが同じで, AがBを含む場合
    Aの先頭とBの先頭が同じで, かつ, 
    Aの末尾をBが含んでおらず, かつ, 
    Aの先頭とBの先頭が同じで, かつ, (重複条件)
    Bの末尾をAが含んでいる
    
(5) AがBを含む場合
    As < Bs かつ, (重複条件)
    Ae > Be かつ, (重複条件)
    Bの先頭をAが含んでいて, かつ
    Bの末尾をAが含んでいる

(6) AとBの末尾アドレスが同じで, AがBを含む場合
    As < Bs かつ, (重複条件)
    Ae == Be かつ, (重複条件)
    Bの先頭をAが含んでいて, かつ
    Bの末尾とAの末尾が等しい

(7) AとBの先頭アドレスが同じで, BがAを含む場合
    ( (4)の領域入れ替え マクロ化すること)
    Aの先頭とBの先頭が同じで, かつ, 
    Aの末尾をBが含んでいる
    Aの先頭とBの先頭が同じで, かつ, (重複条件)
    Bの末尾をAが含んでいない
    
(8) BがAを含む場合
    ( (5)の領域入れ替え マクロ化すること)
    As > Bs かつ, (重複条件)
    Ae < Be かつ, (重複条件)
    Aの先頭をBが含んでいて, かつ
    Aの末尾をBが含んでいる

(9) AとBの末尾アドレスが同じで, BがAを含む場合
    ( (6)の領域入れ替え マクロ化すること)
    As > Bs かつ, (重複条件)
    Ae == Be かつ, (重複条件)
    Aの先頭をBが含んでいて, かつ
    Aの末尾とBの末尾が等しい

(10) Aの末尾がBに接している場合
    As < Bs,                かつ,
    Bの先頭とA末尾+1が等しく, かつ, 
    Bの末尾をAが含んでおらず, かつ, 
    Aの先頭をBが含んでおらず, かつ
    Aの末尾+1とBの先頭が等しい(重複条件)

(11) BよりA全体が前にある場合
    Ae < Bs,                  かつ
    Bの先頭をAが含んでおらず, かつ, 
    Bの末尾をAが含んでおらず, かつ, 
    Aの先頭をBが含んでおらず, かつ
    Aの末尾をBが含んでいない
